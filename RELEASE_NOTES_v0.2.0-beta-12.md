# Release Notes — v0.2.0-beta-12

**Branding, Switch Manager support, and a polished README.**

## Added

- **Integration brand icon and logo** (DALI-2 wordmark with the Lunatone
  crescent), shipped locally with light and dark variants in 256/512 px.
- **Switch Manager support**: a bundled blueprint for the DALI-2 MC 4-button
  coupler plus a setup guide. With Switch Manager installed it auto-deploys to
  `config/blueprints/switch_manager/` and is hash-tracked, so future updates
  refresh old bundled copies while keeping your edits. A normal and a force
  deploy button are available for manual (re)deploy.

## Changed

- README rewritten with a feature overview and sections for brightness,
  turn-on behavior and Switch Manager.

No configuration changes; existing entities and options are unaffected.
